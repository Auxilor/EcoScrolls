## What is EcoScrolls?

EcoScrolls adds custom, config-driven scrolls that inscribe new effects onto items. Use them for item buffs, repeatable modifiers like Hot Potato Books, and drag-and-drop upgrades. It ships with an Inscription Table GUI so players can apply scrolls in game.

## What sets EcoScrolls apart?

Every scroll is a single YAML file built on the shared eco effects system, so the same effects, conditions, and triggers you know from other eco plugins work here too.

- **Built on eco effects:** reuse the effects, conditions, and triggers from across the eco ecosystem.
- **Scroll relationships:** scrolls can require other scrolls or conflict with each other.
- **Type limits:** group scrolls into types and cap how many of each fit on one item.
- **Two ways to apply:** inscribe by drag-and-drop or through the Inscription Table GUI.
